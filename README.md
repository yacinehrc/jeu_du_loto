# Jeu-Du-Loto---Python
Projet transversal qui a comme objectif de créer un jeu de loto en langage python.
<img width="528" height="302" alt="image" src="https://github.com/user-attachments/assets/45af6520-cdf2-4e41-8e56-2742fae4af14" />

Les contraintes étaient :
● Génère un tirage aléatoire de 10 numéros de loto
● Permet à l’utilisateur de saisir ses propres numéros
● Compare les résultats
● calcule un score
● attribue une récompense en fonction du nombre de numéros gagnants

Avant le codes, nous nous sommes renseigné sur les étapes du loto pour pouvoir les réadapter en python.

Le code est construit en fonction pour chaque fonctionnalité du code, pratique pour stoquéer les infomations importantes.
